CSS가 적용된 웹페이지

HTML 구조에 CSS 스타일링이 적용되어 시각적으로 개선된 웹페이지입니다.

배경 (Background)

CSS의 역할

CSS는 HTML 문서에 꾸미기와 배치 조정(스타일과 레이아웃)을 적용하는 언어입니다.

  • 시각적 꾸미기(스타일링)
  • 배치 조정(레이아웃 제어)
  • 화면 크기별 최적화(반응형 디자인)
  • 사용자 경험 개선

효과 (Effects)

보기 좋게 만들기(시각적 개선)

색상, 폰트, 간격 등으로 가독성과 미적 요소를 향상시킵니다.

배치 조정(레이아웃 제어)

요소들의 위치와 크기를 원하는 대로 조정(제어)할 수 있습니다.

일관성 유지

스타일 규칙을 통해 전체 사이트의 디자인 일관성을 유지합니다.

결과 (Results)

CSS를 통해 HTML 뼈대(구조)가 시각적으로 완성됩니다.

HTML + CSS 단계의 특징
항목 상태 설명
시각적 표현 뛰어남 현대적인 디자인과 효과 적용
상호작용 제한적 호버 효과는 있으나 동적 기능 없음
로딩 속도 빠름 CSS 파일 하나만 추가로 로드
접근성 우수 시맨틱 구조 유지하며 시각적 개선

한계 (Limitations)

  1. 정적 상호작용

    CSS만으로는 복잡한 사용자 상호작용이나 동적 기능을 구현할 수 없습니다.

  2. 데이터 처리 불가

    사용자 입력을 처리하거나 서버와 통신하는 기능이 없습니다.

  3. 상태 관리 한계

    페이지의 상태를 기억하거나 변경할 수 없습니다.

  4. 이벤트 처리 부재

    클릭, 스크롤 등의 사용자 이벤트에 대한 복잡한 응답이 불가능합니다.
